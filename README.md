## 💡 Introduction

The **Masjid Donation Management System** is a web-based system developed to simplify and digitalize the process of managing **donations (tabung)** and **disbursements** in a mosque environment.

This system aims to improve efficiency, transparency, and organization by replacing manual processes with a structured digital platform.

---

## 📌 System Overview

* **System Name:** Masjid Donation Management System
* **Developed For:** Masjid Bandar Rinching
* **Purpose:**
  To manage donations, disbursement requests, and reporting efficiently in a centralized system

---

## 👥 Users

* 🧑‍💼 **President Masjid**
* 🧑‍🤝‍🧑 **AJK Masjid (Committee Members)**
* 🙋 **Donors (Community)**

---

## ✨ Key Features

### 🧑‍💼 President (Admin)

* Manage **tabung (funds)**:

  * Create
  * Edit
  * Delete
* Approve or reject **disbursement requests**
* Create accounts for **AJK Masjid**
* Generate **reports and summaries**
* Filter reports by **date**

---

### 🧑‍🤝‍🧑 AJK Masjid

* Submit **disbursement requests**
* View request status
* Edit or delete requests
* Filter requests by **purpose**
* Update profile
* View summary

---

### 🙋 Donors

* Make **donations**
* View donation **slips**
* Download donation **receipts**

---

## 🛠️ Technologies Used

* **Database:** SQLite

## 🚀 Conclusion

The **Masjid Donation Management System** successfully meets most of its intended objectives by providing:

* ✅ Efficient donation management
* 🔒 Secure access control
* 📊 Clear reporting and tracking
* 👥 Better coordination between President, AJK, and donors

Although some advanced features were not implemented, the system delivers a **solid, functional foundation** that can be expanded in the future.
